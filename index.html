Lists

We have 3 important types of lists, we are going to take a look through the 3 types. I can do a list with breaks < br >:
   First item
   Second item
   Third item
But this is not a list... we should use list items in a ordered list:<ol> <li>:

  1. ... First item
  2. ... Second item
  3. ... Third item

So we have seen how to make an ordered list, to make a unordered list we write ul instead of ol:

  • ... item
  • ... item
  • ... item

If we want to add extra space betweem items we write each item in a paragraph enviorment. Finally we have the description list, we can expect how to create this list.
